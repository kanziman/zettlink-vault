Someone just open sourced a harness that builds real robots with AI.

Designing mechanical parts used to mean stitching together half a dozen tools. 

Now you can describe a robot arm in plain text and watch it appear.

A new open source harness lets coding agents generate real 3D CAD models from natural language. 

You describe the part. 

The agent writes the source files. 

The harness builds the geometry.

One demo shows a 7 degree-of-freedom robot arm with working kinematics and a custom GUI, generated almost entirely through prompts.

What ships with the repo:

> Exports STEP, STL, DXF, GLB
> Produces URDF robot descriptions
> Local viewer for inspecting geometry
> Stable handles for precise follow-up edits
> Runs fully offline, no backend

The trick is source-controlled files. 

Agents edit code, not opaque binaries, so every change is reviewable and reproducible.

Weeks of tool-juggling collapse into a single conversation loop.
