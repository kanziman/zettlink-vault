클로드 코드(Claude Code) 세팅하는 데 30분 쓰지 마세요. 이제 3분이면 끝납니다.
지난번 공유드렸던 ‘클로드 코드 원격 제어 + tmux 멀티 패널’ 환경, 약속대로 오픈소스로 배포했습니다. 이름은 batipanel입니다.

10년 넘게 자동화를 해오면서 제가 가장 편하게 느끼는 터미널 작업 환경을 도구화했습니다. 저 스스로가 이 환경 없이는 코딩이 안 될 정도라, 현재 클라이언트 프로젝트와 batipanel 자체 개발도 이 위에서 하고 있습니다.

핵심은 간단합니다. "귀찮은 건 AI가, 세팅은 batipanel이 합니다"
그동안 클로드 코드 CLI로 개발하려면 매번 뭘 하셨나요?
- Node, npm 설치 및 버전 관리
- Claude Code 설치 및 인증
- Oh My Zsh, Powerline 테마 세팅
- tmux 수동 설정, 패널 구성
- 작업환경 세션 관리

이 지루한 과정을 스크립트 한 줄이 전부 처리합니다. npx batipanel 또는 curl 한 줄이면 설치부터 환경 구성까지 완료됩니다. 실행은 터미널에서 b 한 글자만 치면 끝입니다. 화면 크기와 작업 스타일에 맞춰 최적의 레이아웃(AI 코딩 + Git + 모니터링 + 탐색기)이 자동 구성됩니다.

이런 환경이 필요하셨던 분들께 특히 유용합니다. 
1) 맥북/아이패드/모바일 어디서든 작업하고 싶은 분
서버에 batipanel을 띄우면 tmux 세션이 24시간 유지됩니다. 별도의 리모트 기능 없이도 언제 어디서든 터미널 앱으로 접속해 어제 하던 작업을 100% 그대로 이어갈 수 있습니다.

2) AI 코딩의 몰입감을 극대화하고 싶은 분
AI가 코드를 짜는 동안 시스템 자원을 모니터링하고, Git 상태를 확인하며, 파일 구조를 한눈에 보는 'AI 네이티브' 환경이 펼쳐집니다.

3) 퇴근 후 다시 세팅하기 싫은 분
다음 날 아침 "b myproject" 한 줄이면 어제의 작업 맥락이 그대로 살아납니다.

아직 모든 환경에서 테스트가 완료된 건 아니라서, 여러분의 개발 환경(OS, 쉘 등)을 댓글로 알려주시면 빠르게 대응해서 업데이트하겠습니다.

오픈소스 주소와 상세 가이드는 첫 번째 댓글에 남겨두겠습니다.
도움이 되셨다면 GitHub에 ★ 별하나로 응원해 주세요. 더 많은 분들께 알릴 수 있습니다
