"매일 9시에 홍길동한테 온 카톡 읽고 요약해서 김팀장한테 보내줘."
이 한마디에 AI가 직접 카카오톡을 열고, 메시지를 읽고, 요약한 뒤 전달까지 합니다.

지난주 무료 공개했던 카카오톡 자동화 앱 BatiFlow에 보내주신 반응에 힘입어, 일주일 만에 대폭 업그레이드해서 다시 배포합니다. 이번 업데이트의 핵심은 단순 발송을 넘어, AI 에이전트를 macOS 전체로 확장했다는 점입니다. 

- 이전: 카카오톡 메시지 보내기 
- 이번: AI가 50개+ 도구를 스스로 선택하고 실행하는 OS 에이전트

이제 이런 명령이 실제로 가능해집니다.
- "사내 어드민에서 어제 매출 캡처해서 팀 카톡에 공유해줘."
- "구글 뉴스에서 'AI 자동화' 검색해서 캡처하고 내 메모앱에 저장해줘."
- "다운로드 폴더 PDF 요약해서 내일 할 일로 등록해줘."
- "내일 오후 3시에 회의 잡아줘."

왜 BatiFlow일까? 
OpenClaw 등 강력한 오픈소스 에이전트들이 쏟아지고 있지만, 비개발자가 실무에 쓰기엔 여전히 장벽이 높습니다. 저희는 이 복잡한 기술을 GUI로 감싸서, 누구나 클릭 몇 번으로 맥(macOS)의 네이티브 앱들을 자연어로 제어하게 만들었습니다.

최근 빌더 조쉬 Josh Kim님과의 인터뷰에서 오픈소스 언어모델과 보안 이슈에 대해 말씀을 나눈 적 있는데, BatiFlow는 Ollama를 통해 로컬에서 AI를 실행할 수 있어, 민감한 데이터를 외부로 보내지 않고 사용할 수 있습니다. 앱을 설치하면 맥 사양을 감지해서 최적의 오픈소스 모델을 추천해주고, 원클릭으로 설치까지 끝납니다. 터미널이나 복잡한 설정 없이 바로 시작할 수 있습니다. 

주요 특징:
1. macOS 앱 제어: 브라우저, Finder, 캘린더, 메일, 메모, 미리알림 통합 관리
2. AI 커맨드 패널: ⌘⌥Space 한 번으로 어디서든 AI 호출 가능
3. 비용 부담 제로: 로컬 모델 무료 사용, Google Gemini 무료 API 지원, 유료 API 사용량 실시간 확인
4. 워크플로우 자동 생성: 화면 녹화로 업무 기록, AI가 반복 가능한 스킬로 변환 후 스케줄링

"Work Less, More Impact" 
좋은 기술을 누구나 쉽게 쓸 수 있도록 만드는 것이 저희 바티(Bati) 팀의 철학입니다. 내부 RPA 기술과 Agent AI 노하우를 집약해 만든 이 도구를 무료로 나눕니다. 기술은 나눌수록 더 커진다고 믿기 때문입니다.

맥북을 쓴다면 BatiFlow 깔아두세요. 무료 로컬 AI + 카톡 자동화 + OS 에이전트, 이 셋이 한 앱에 들어있습니다. 직접 써보시고 피드백 주시면 적극 반영하겠습니다.
