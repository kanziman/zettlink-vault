🚨 GitHub이 개인정보 정책을 변경했습니다.

4월 24일부터 자사 AI Copilot 학습에 사용자의 데이터를 활용할 수 있습니다.
비공개 데이터 학습을 원치 않는다면, 지금 비활성화 하셔야합니다.

---

지난 25일, GitHub은 GitHub Copilot의 상호작용 데이터를 자사 및 Microsoft의 AI 모델 학습에 사용할 예정임을 밝혔습니다. 수집되는 데이터는 Copilot을 통한 입력, 출력, 코드, 기타 컨텍스트 등이 포함됩니다. 이 정책은 개인용 요금제에만 해당되고, 교육/Business/Enterprise 요금제 사용자에게는 해당되지 않습니다.

기존 Copilot 모델은 public 코드만을 활용해 만들어졌지만, 더 똑똑한 모델을 만들기위해 실제 사용자의 사용 패턴, 입력 데이터 등을 추가 학습할 필요를 느꼈다고 합니다.

Copilot에게 입력된 프롬프트, Copilot이 작업과정에서 읽은 모든 코드가 학습 대상으로 포함됩니다. 특히 vscode에 연동해두고 tab completion 이용중이신분들은 주의하셔야겠습니다. API key, token, 비밀번호와 같은 민감정보는 미리 제거하도록 여러 겹의 안전장치를 준비했다고합니다. 

---

커뮤니티 반응은 좋지 않습니다.

"수집되기 싫으면 빨리 비활성화하라고 하는건 다크패턴이다. 수집에 동의하면 활성화 해달라고 공지하는 것이 맞지 않나?" 와 같은 의견이 지배적인데요,

대부분의 개발자들이 코드를 보관하고있는 GitHub인만큼, Privacy 침해 우려에 대한 비판도 거센 상황입니다. 

---

비활성화는 GitHub 홈페이지의 settings 메뉴에서 할 수 있습니다.

1. 다음 링크로 접속 (https://lnkd.in/g6_CDEGR)
2. AI model training - Disabled 설정

학습은 오는 4월 24일부터 시작된다고 하니,
원치 않는 분들은 미리 비활성화 해주시면 됩니다!

읽어주셔서 감사합니다. 🙇‍♂️
