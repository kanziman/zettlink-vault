코딩과 에이전트 성능에서 Gemma4 보다 더 뛰어난 모델이 나왔습니다. 지금 바로 맥(macOS)에서 구축할 수 있는 버전을 무료 공개합니다.

어제 새벽 알리바바가 공개한 Qwen 3.6을 바티팀이 실기기 검증을 마치고 양자화하여, BatiFlow에서 클릭 몇 번으로 바로 쓸 수 있도록 오픈했습니다.

이 모델이 주목받는 이유는 두 가지입니다.

첫째, 속도. 
35B 파라미터 중 3B만 활성화하는 구조라 모델 크기 대비 속도가 놀라울 정도로 빠릅니다. 맥 메모리만 넉넉하면 클라우드 못지않은 체감 속도가 나옵니다. 
허깅페이스(댓글)에 실제 맥에서 구동되는 속도를 영상으로 올려두었으니 직접 확인해보세요.

둘째, 성능. 
지난주 공개된 구글 Gemma 4를 코딩과 에이전트 영역에서 확실히 앞섭니다.
- 코딩 에이전트(SWE-bench): 73.4 vs 52.0
- MCP 도구 활용(MCPMark): 37.0 vs 18.1
- 수학 추론(AIME 2026): 92.7 vs 89.2

Apache 2.0 라이선스로 누구나 무료로 쓸 수 있고, 바티 최적화를 통해 16GB 맥에서도 일단 구동은 가능한 수준까지 메모리를 낮췄습니다. 메모리 효율을 최적화한 양자화 모델을 제작하고, 도구 호출(Function Calling)과 한국어 답변까지 실디바이스에서 검증 완료했습니다.

BatiFlow 앱이면 설치부터 실행까지 한 번에 끝나고, 개발자분들은 Ollama 한 줄이면 됩니다. 세계적으로 핫한 오픈 모델이 나올 때마다, 양자화 모델을 공개하고 BatiFlow에도 바로 반영할 예정입니다.
