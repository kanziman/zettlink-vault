카카오톡, Slack, iMessage — API 없이 자동 발송이 가능한 macOS 앱을 무료로 배포합니다.

"API도 없는 일반 채팅방인데 자동화가 되나요?" 네, 됩니다. 저희 바티팀이 1년 넘게 고도화해 온 AI 워크플로 자동화에서 마지막 퍼즐은 카카오톡이었습니다. 카톡은 일반 채팅용 API를 제공하지 않지만, 많은 고객사가 보안과 편의상 일반 채팅방을 선호합니다. 그래서 macOS의 Accessibility API를 활용해 AI가 직접 OS 레벨에서 카카오톡 앱을 제어하도록 설계했습니다. 네이티브 앱이라 용량도 3MB가 안 됩니다.

이제 Claude Code나 Cursor에서 작업이 끝나면 딱 한 줄만 입력합니다. 
"김 대표님께 카톡으로 이번 수정 완료 사항 정리해서 보내줘." 
5초 뒤, 실제 카카오톡 앱이 움직이며 메시지가 전송됩니다.

최근 빌더 조쉬 (Josh Kim)님과의 인터뷰에서 저희 팀의 AI 워크플로를 소개한 적 있는데, BatiFlow도 그 흐름의 일부입니다.

- 접수: 고객 요청 시, AI가 과거 이력을 기반으로 요약, 예상 범위, 예상 소요 시간까지 사전 정리해 빠른 초기 대응을 지원합니다.
- 처리: 기획자와 개발자는 AI의 분석을 활용하면서 판단과 품질 관리에 집중하는 오케스트레이션 역할을 수행합니다.
- 완료: 배포가 끝나면 내부 슬랙 공유, 고객사 게시판 상태 업데이트, 댓글 안내, 메신저 전달까지 자동으로 이어집니다.

도움을 받은 그 이상으로 돌려드리고 싶습니다. AI 시대에는 훌륭한 오픈소스가 정말 많고, 저희 팀도 그 혜택을 입었습니다. 개발자가 아닌 분들도 클릭 몇 번으로 이 효율을 누리실 수 있도록 GUI 앱으로 패키징했습니다.

[BatiFlow 주요 기능]
- 메신저 통합: 카카오톡 / Slack / iMessage 자동 발송 지원
- MCP 연동: Claude Code, Cursor에서 원클릭 설정 (환경설정에서 즉시 가능)
- 개인화 메시지: 여러 명 선택 시 {name} 치환 기능으로 맞춤 메시지 일괄 발송
- 스크립트 실행: CLI에서 직접 호출 가능, 기존 자동화 파이프라인에 바로 연결
- 안전장치: 오발송 방지를 위한 사전 확인 및 제어 로직 내장

기술은 나눌수록 커진다고 믿습니다. 직접 써보시고 피드백 주시면 적극 반영하겠습니다.
